## ‚ö° Quick Steps (TL;DR)

1. **Push to GitHub** ‚Üí Create repo, push code
2. **Railway (Backend)** ‚Üí Deploy from GitHub, add DB, set env vars
3. **Vercel (Frontend)** ‚Üí Deploy from GitHub, set `REACT_APP_API_URL`
4. **Connect** ‚Üí Update Railway `CORS_ORIGINS` with Vercel URL

---

## üîë Key URLs & Settings

### Railway Backend Settings:
- **Root Directory**: `backend`
- **Start Command**: `uvicorn app.main:app --host 0.0.0.0 --port $PORT`

### Vercel Frontend Settings:
- **Root Directory**: `./frontend` (if monorepo) OR `./` (if separate repo)
- **Build Command**: `npm run build`
- **Output Directory**: `build`

---

## üìù Environment Variables

### Railway (Backend):
```
OPENAI_API_KEY=your-key-here
OPENAI_MODEL=gpt-4o
DATABASE_URL=postgresql://... (auto-generated by Railway)
SECRET_KEY=generate-with-python-secrets-token_urlsafe-32
ENVIRONMENT=production
CORS_ORIGINS=https://your-app.vercel.app
```

### Vercel (Frontend):
```
REACT_APP_API_URL=https://your-backend.railway.app/api
```

---

## üîó URL Format

- **Backend API**: `https://your-app.railway.app/api`
- **Frontend**: `https://your-app.vercel.app`
- **CORS**: Must match frontend URL exactly (including `https://`)

## ‚úÖ Checklist

### Before Deploying:
- [ ] Code works locally
- [ ] Git repo created and pushed
- [ ] OpenAI API key ready
- [ ] Secret key generated

### Railway (Backend):
- [ ] Project created from GitHub
- [ ] Root directory set to `backend`
- [ ] Start command configured
- [ ] PostgreSQL database added
- [ ] All environment variables set
- [ ] Database migration run
- [ ] Backend URL copied

### Vercel (Frontend):
- [ ] Project imported from GitHub
- [ ] Root directory configured
- [ ] `REACT_APP_API_URL` environment variable set
- [ ] Build successful
- [ ] Frontend URL copied

### Connection:
- [ ] Railway `CORS_ORIGINS` updated with Vercel URL
- [ ] Test login works
- [ ] API calls succeed
- [ ] No console errors

---

## üêõ Quick Fixes

| Problem | Solution |
|---------|----------|
| CORS Error | Update `CORS_ORIGINS` in Railway with exact Vercel URL |
| API 404 | Check `REACT_APP_API_URL` includes `/api` |
| Build Failed | Check root directory and build command |
| DB Error | Verify `DATABASE_URL` in Railway variables |
| 500 Error | Check Railway logs for specific error |
